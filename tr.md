# Броня
Позволяет сдержать %Урона противника. Каждая броня требует характеристик.
Каждая броня накладывает свой дебаф либо к стамине либо к Здоровью или к Оружию

# Зелья
Зелья тратят стамину на использование. Каждое зелье накладывает свой эффект 


# Уровень и опыт
Опыт - Выдается за убийство. Нужен для прокачки уровня
Уровень - Позволяет прокачивать персонажа. Чем больше уровень тем больше нужно опыта для
прокачки


[Предметы]
# Оружие
Название - Описание - Урон - Шанс попадания, Требования: С - .. Л - ..

# Броня
Название - Описание - %Сдерживания урона, Требования: С - .. Л - .. , Дебаф - Ст. Зд. т.д.

# Зелья
Название - Описание - Эффект: +Зд. +Ст. +Урон. +Защита от огня

[Эффекты]
Эффекты это дополнительный префикс который может очень сильно влиять на игру

Эффекты для оружия:
1. Огонь - Наносит каждый ход по 1 урону. Длительность 3 хода
2. Холод - Снимает каждый ход по 1 стамине. Длительность 2 хода
